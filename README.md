## Intro

This repository provides pre-built binaries (`.xcframework`) of original repository [Hyperion-iOS](https://github.com/willowtreeapps/Hyperion-iOS). The original Hyperion-iOS repository does not supports SPM, I built the library to `.xcframework` so that it could be used in SPM
